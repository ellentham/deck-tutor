# Modern Format Rules

Use this context when recommending cards for Modern decks.

## Deck Structure

- **60 cards** minimum in the main deck
- **15 cards** in the sideboard (optional but standard)
- **Up to 4 copies** of any card (except basic lands)

## Format Definition

- Modern includes all Standard-legal sets from **Eighth Edition** (2003) onward
- Cards are legal if printed in a Modern-legal set, regardless of printing
- New sets become legal on their release date

## Ban List

- Modern has a format-specific ban list
- Check current legality before recommending

## Common Deckbuilding Ratios

- **Lands**: 22–24 (aggro) to 26–28 (control)
- **Mana curve**: Heavily format-dependent; Modern is generally faster than Standard
- **Interaction**: Maindeck removal and counters; sideboard for specific matchups

## Metagame Considerations

- Modern is a diverse, non-rotating format
- Meta shifts with new set releases and ban list updates
- Consider: linear strategies vs. interactive decks, combo vs. aggro vs. midrange
