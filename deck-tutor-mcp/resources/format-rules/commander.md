# Commander Format Rules

Use this context when recommending cards for Commander (EDH) decks.

## Deck Structure

- **99 cards** in the main deck + **1 commander**
- **Singleton**: No more than one copy of any card (except basic lands)
- **100-card total** including the commander

## Commander Rules

- Commander must be a **legendary creature** (or have "can be your commander")
- Commander determines **color identity**: all cards in the deck must only use mana symbols that appear on the commander (including reminder text, cost, and rules text)
- Commander starts in the **command zone** and can be cast from there
- If the commander would go to graveyard or exile, it may be returned to the command zone instead (increasing cost by {2} per previous cast each time)

## Color Identity

- A card's color identity is all mana symbols that appear anywhere on the card
- Hybrid mana (e.g. {W/U}) counts as both colors
- Phyrexian mana counts toward color identity
- Colorless cards (no colored symbols) can go in any deck

## Ban List

- Commander has a format-specific ban list (separate from Vintage/Legacy)
- Check current legality before recommending

## Common Deckbuilding Ratios

- **Lands**: ~36–40 (adjust for ramp and curve)
- **Ramp**: 8–12 (mana rocks, dorks, land ramp)
- **Card draw**: 8–12
- **Interaction**: 5–10 (removal, counterspells)
- **Win conditions / threats**: Remainder

## Power Level Considerations

- **Casual**: Themed, flavorful, lower consistency
- **Focused**: Optimized for a strategy, fewer pet cards
- **Optimized**: High efficiency, fast mana, tutors
- **Competitive**: Maximum power, combo-oriented
