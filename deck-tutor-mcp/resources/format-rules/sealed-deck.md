# Sealed Deck Format Rules

Use this context when recommending or evaluating cards for Sealed Deck.

## Deck Structure

- **40 cards** minimum in the main deck
- Built from **6 booster packs** (typically 90 cards) plus unlimited basic lands
- No sideboard in the traditional sense—any opened cards can be swapped between games

## Format Definition

- **Limited format**: Build a deck on the spot from opened packs
- **1v1** games
- Format featured at **Prerelease** events for each new set
- Card pool varies by set; check which set is being used

## Play Rules

- One-on-one games after deck building
- Best of one or best of three
- ~20 minutes average game length

## Sealed Evaluation Priorities

- **BREAD**: Bombs, Removal, Evasion, Aggro, Duds
- **Mana curve**: Prioritize 2–4 mana plays
- **Color commitment**: Typically 2 colors; sometimes 3 if splashing for bombs
- **Splash**: Splash only for powerful cards (1–3 mana symbols of splash color)
- **Playables**: Count how many playable cards you have per color

## Deckbuilding Ratios (40-card deck)

- **Lands**: 16–18 (typically 17)
- **Creatures**: 14–18
- **Spells**: 6–10 (removal, combat tricks, card draw)

## Metagame Considerations

- No drafting—you only see your own cards
- More variance than draft; harder to build a cohesive deck
- Prerelease often uses 6 packs of the new set
- Good for learning a new set
