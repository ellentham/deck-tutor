# Standard Format Rules

Use this context when recommending cards for Standard decks.

## Deck Structure

- **60 cards** minimum in the main deck
- **15 cards** in the sideboard (optional but standard)
- **Up to 4 copies** of any card (except basic lands)

## Format Definition

- Standard uses cards from the **most recently released Magic sets**
- **Rotating format**: Each year, after the fall set Prerelease, the four oldest sets rotate out
- New sets become legal on their release date
- Check the [current Standard legal sets](https://magic.wizards.com/en/formats/standard) and [banned list](https://magic.wizards.com/en/banned-restricted-list#standard-banned)

## Play Rules

- One-on-one games
- Best of one or best of three
- ~20 minutes average game length

## Common Deckbuilding Ratios

- **Lands**: 22–26 (adjust for curve and mana requirements)
- **Mana curve**: Tends to be lower than eternal formats; games are shorter
- **Interaction**: Maindeck removal and counters; sideboard for specific matchups

## Metagame Considerations

- Meta shifts with each new set release and rotation
- Accessible card pool—easier to collect than eternal formats
- Competitive pathway to Arena Championships and Pro Tour
