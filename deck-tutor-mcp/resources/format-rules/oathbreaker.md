# Oathbreaker Format Rules

Use this context when recommending cards for Oathbreaker decks.

## Deck Structure

- **60 cards** total (58 + 1 Oathbreaker + 1 signature spell)
- **1 Oathbreaker** (planeswalker that serves as commander)
- **1 signature spell** (instant or sorcery in your Oathbreaker's color identity)
- **Singleton**: No more than one copy of any card (except basic lands)
- **15 cards** in the sideboard (optional)

## Format Definition

- Commander-style format with a **60-card deck**
- Uses a **planeswalker** as commander (Oathbreaker) plus a **signature spell**
- Both Oathbreaker and signature spell start in the command zone
- Signature spell can only be cast when the Oathbreaker is on the battlefield
- Check the [banned list](https://magic.wizards.com/en/banned-restricted-list) for Oathbreaker

## Oathbreaker Rules

- Oathbreaker must be a **planeswalker**
- Signature spell must be an **instant or sorcery** within the Oathbreaker's color identity
- Both can be cast from the command zone
- If either would go to graveyard or exile, it may return to the command zone (cost increases by {2} per previous cast each time)

## Play Rules

- Typically multiplayer (3–4 players) or 1v1
- Starting life total: 20 (multiplayer) or 30 (1v1)

## Common Deckbuilding Ratios

- **Lands**: ~24–28
- **Ramp**: 6–10
- **Card draw**: 6–10
- **Interaction**: 4–8

## Metagame Considerations

- Faster than Commander due to smaller deck and planeswalker focus
- Signature spell adds consistency and combo potential
- Popular as a casual format
