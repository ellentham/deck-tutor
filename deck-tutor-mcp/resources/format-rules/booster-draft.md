# Booster Draft Format Rules

Use this context when recommending or evaluating cards for Booster Draft.

## Deck Structure

- **40 cards** minimum in the main deck
- Built from **~45 drafted cards** (3 booster packs) plus unlimited basic lands
- No sideboard in the traditional sense—unused drafted cards can be swapped between games

## Format Definition

- **Limited format**: Build a deck on the spot from opened packs
- **2–8 players** typically; 8 is standard for competitive draft
- Each player opens 3 booster packs, picks one card at a time, passes the rest
- Pack 1: pass left. Pack 2: pass right. Pack 3: pass left.
- Card pool varies by set; check which set is being drafted

## Play Rules

- One-on-one games after deck building
- Best of one or best of three
- ~20 minutes average game length
- ~120 minutes total including draft and deck building

## Draft Evaluation Priorities

- **BREAD**: Bombs, Removal, Evasion, Aggro, Duds
- **Mana curve**: Prioritize 2–4 mana plays; avoid too many high-cost cards
- **Color commitment**: Stay open early; commit to 2 colors after pack 1
- **Synergy**: Cards that work well together in the set
- **Signal reading**: What colors are open based on what you receive?

## Deckbuilding Ratios (40-card deck)

- **Lands**: 16–18 (typically 17)
- **Creatures**: 14–18
- **Spells**: 6–10 (removal, combat tricks, card draw)

## Metagame Considerations

- Set-specific; each set has different mechanics and power levels
- Format featured at prerelease and FNM
- Level playing field—no pre-built deck required
